# Скрипты для Базис-Мебельщик (ПрофиГрупп)

Набор модульных скриптов для автоматизации работы конструктора в Базис-Мебельщик. Проект включает в себя удобный графический интерфейс с поиском и разделением функций по категориям.

## 🚀 Основные возможности

Интерфейс разделен на логические блоки:
*   **Переименование**: Массовое изменение имен панелей, замена слов, работа с регистром.
*   **Фурнитура**: Скрытие крепежа, удаление пустых схем, проверка отверстий.
*   **Модель**: Проверка кромок, скриншоты модели, работа с цветами материалов.
*   **Выделение**: Умный поиск объектов по позициям, обозначениям или свойствам.
*   **Панели**: Расчет веса, площади и параметров разделителей.

## 🛠 Технологии и структура

Проект построен по модульному принципу:
*   `main.js` — точка входа (диспетчер).
*   `config.js` — конфигурация кнопок и групп.
*   `ui_core.js` — ядро графического интерфейса с функцией живого поиска.
*   `logic_*.js` — изолированные модули с бизнес-логикой скриптов.
*   `utils.js` — общие вспомогательные функции (бэкапы, алерты).
